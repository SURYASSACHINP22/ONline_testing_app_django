"# Online Testing System (Django)

A simple online testing system: candidates register, take tests, view results and history, manage membership plans, and can delete their account. Admins use Django Admin to see all users, delete accounts, and assign membership plans.

## Design patterns

See **[DESIGN_PATTERNS.md](DESIGN_PATTERNS.md)** for the patterns used. **How to log in to admin:** see **[ADMIN_LOGIN.md](ADMIN_LOGIN.md)** (create superuser, then open `/admin/`).
